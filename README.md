# Hyperlocal Alerts & Marketplace

A Next.js + Node.js application for hyperlocal alerts and community marketplace.

## Features
- **Alerts:** specific to user location.
- **Marketplace:** Buy and sell items within a 5km radius.
- **Tech Stack:** Next.js (Frontend), Express (Backend), MongoDB (Database).

## Getting Started

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```
